B1: Tạo file backup thư viện: requirements.txt
+ Cài đặt thư viện từ requirements:
# pip install -r requirements.txt

+ Lưu các thư viện trong dự án vào requirements.txt:
# pip freeze > requirements.txt

B2: Tạo virtualenv
# virtualenv env

B3: Kích hoạt virtualenv
# ./env/Scripts/activate

B4: Cài đặt thư viện trong virtualenv
# pip install selenium (ví dụ)

B5: Huỷ kích hoạt virtualenv
# deactivate
